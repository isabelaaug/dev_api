## Descrição do projeto :page_facing_up:

API desenvolvida para demonstrar as diferenças entre utilizar com Django o Flask e Flask-RESTful.
Exemplo de REST API para manipular dados referentes ao registro de desenvolvedores e suas habilidades.

## Funcionalidades :gear:

- GET - Exibir todos desenvolvedores e suas habilidades cadastradas
- POST - Cadastrar desenvolvedores e suas habilidades
- PUT - Atulizar dados de desenvolvedor (buscando pelo ID)
- DELETE - Excluir registro de desenvolvedores (buscando pelo ID)

## Linguagens, dependencias e libs utilizadas :books:

- Python
- Django
- API REST
- Flask
- Flask-RESTful
